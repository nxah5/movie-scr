# 스크립트 용지 배포 폴더

이 폴더는 온라인 정적 호스팅에 바로 업로드할 수 있는 배포본입니다.

## 포함 파일

- `index.html`: 실제 앱 파일입니다. 서버에서 기본 페이지로 열립니다.
- `README.md`: 배포 메모입니다.

## 업로드 방법

Netlify, Vercel, GitHub Pages, Cafe24, 일반 웹호스팅 등에서 이 폴더의 내용을 그대로 업로드하면 됩니다.

빌드 과정이나 패키지 설치는 필요 없습니다.

## 주의

입력한 데이터는 브라우저의 `localStorage`에 저장됩니다. 같은 브라우저에서는 새로고침 후에도 유지되지만, 다른 기기나 다른 브라우저로 자동 동기화되지는 않습니다.
